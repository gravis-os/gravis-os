# GravisOS CLI

A CLI for gravis.

## Installation
`npm install -g @gravis-os/cli`

or 

`npx @gravis-os/cli <COMMAND>`

## Usage

`gravis --help`

## Customizing your CLI

Check out the documentation at https://github.com/infinitered/gluegun/tree/master/docs.

## Publishing to NPM

To package your CLI up for NPM, do this:

```shell
$ npm login
$ npm whoami 
$ npm test

$ npm run build

$ npm publish
```
